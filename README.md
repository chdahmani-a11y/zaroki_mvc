#  Club Scientifique - Gestion des Membres (MVC + Flask + Pandas)

Projet réalisé dans le cadre du module **Architecture MVC**.  
Ce projet permet de gérer les membres d’un club scientifique universitaire avec :

- Lecture / stockage des données dans des fichiers CSV
- Interface Console (CLI)
- Interface Web avec Flask
- Recherche par compétence
- Visualisation des statistiques avec Pandas + Matplotlib
